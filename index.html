ğŸ—’ï¸ From where do I get the data?

The data for the family tree is stored in two CSV-files:

  • one containing all people
  • one containing the connections between them

If someone shared their family tree with this link, you should have received those two files with it. Just upload them in the form above.
If instead you want to create your own family tree, you need to create those files for yourself.

Each CSV-file is a table with different columns, depending on the type. You can create those in a spreadsheet program of your choice and then export it as CSV.

Information on how to create those files is available in the wiki on GitHub.
